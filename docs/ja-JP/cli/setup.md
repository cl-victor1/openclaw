# `openclaw setup`

`~/.openclaw/openclaw.json` とエージェントワークスペースを初期化します。

関連:

- はじめに: [はじめに](/start/getting-started)
- ウィザード: [オンボーディング](/start/onboarding)

## 使用例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

セットアップからウィザードを実行する場合：

```bash
openclaw setup --wizard
```
