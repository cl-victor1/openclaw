# スクリプト

`scripts/` ディレクトリには、ローカルワークフローや運用タスク用のヘルパースクリプトが含まれています。
タスクが明確にスクリプトに関連付けられている場合にこれらを使用してください。それ以外の場合はCLIを優先してください。

## 規約

- スクリプトは、ドキュメントやリリースチェックリストで参照されていない限り**オプション**です。
- CLI画面が存在する場合はそちらを優先してください（例：認証モニタリングは `openclaw models status --check` を使用します）。
- スクリプトはホスト固有であると想定してください。新しいマシンで実行する前に内容を確認してください。

## スクリプトを追加する場合

- スクリプトは焦点を絞り、ドキュメント化してください。
- 関連するドキュメントに短いエントリを追加してください（存在しない場合は作成してください）。
